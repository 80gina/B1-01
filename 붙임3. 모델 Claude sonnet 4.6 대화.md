* 모든 레시피에 "이것만 주의하면 돼\!" 포인트 1줄 추가  
* 초보자가 흔히 하는 실수 미리 알려주기

### 3️⃣ 가족 맞춤 구성

* 아이 입맛 / 어른 입맛 동시 공략 팁 제공  
* 1인분 → 4인분 양 조절 공식 수록

## 🛒 재료 (2인분 기준)

### 핵심 재료

| 재료 | 용량 | 구매 팁 |
| ----- | ----- | ----- |
| 소면 | 200g (2줌) | 마트 기본 소면 OK |
| 계란 | 2개 | 냉장 보관 것 사용 |
| 간장 | 3큰술 | 국간장 말고 진간장 사용 |

### 양념 재료

| 재료 | 용량 | 없으면? |
| ----- | ----- | ----- |
| 참기름 | 1큰술 | 들기름으로 대체 가능 |
| 설탕 | 1작은술 | 올리고당으로 대체 가능 |
| 다진 마늘 | 0.5작은술 | 생략 가능 |
| 고춧가루 | 1작은술 | 안 매운 버전은 생략 |
| 통깨 | 약간 | 생략 가능 |

### 선택 재료 (있으면 더 맛있어요\!)

code

📋 복사

🥒 오이 1/3개    → 채 썰어 올리면 아삭한 식감 추가  
🌿 쪽파 2줄기    → 송송 썰어 올리면 향 UP

🥕 당근 1/4개    → 채 썰어 올리면 색감 UP

## 🍳 필요한 도구

code

📋 복사

## 👨‍🍳 조리 순서 (총 5분)

### ⏱️ STEP 1 (0분\~1분) — 물 올리기 \+ 양념장 만들기

💡 핵심 포인트 : 물 끓이는 동안 양념장을 만들어야 시간을 아낄 수 있어요\!

code

📋 복사

1\. 냄비에 물 가득 붓고 센 불로 올리기  
   → 물의 양 : 냄비의 70% 정도  
   → 소금 한 꼬집 넣으면 면이 더 쫄깃해져요\!

2\. 기다리는 동안 볼에 양념장 만들기  
   → 간장 3큰술  
   → 참기름 1큰술  
   → 설탕 1작은술  
   → 다진 마늘 0.5작은술  
   → 고춧가루 1작은술

   → 젓가락으로 잘 섞어주기

---

### ⏱️ STEP 2 (1분\~2분) — 계란 프라이 만들기

💡 핵심 포인트 : 계란은 반숙으로 만들어야 비빌 때 소스처럼 퍼져서 더 맛있어요\!

code

📋 복사

1\. 프라이팬을 중불로 예열 (30초)

2\. 식용유 살짝 두르기  
   → 키친타월로 얇게 펴 바르면 기름 덜 써도 OK

3\. 계란 깨서 넣기  
   → 너무 센 불 ❌ → 흰자 타고 노른자 굳어버림  
   → 중불 유지 ✅

4\. 흰자가 하얗게 익으면 불 끄기  
   → 노른자는 살짝 흔들렸을 때 완성

   → 뚜껑 덮으면 더 빨리 익어요\!

---

### ⏱️ STEP 3 (2분\~4분) — 소면 삶기

💡 핵심 포인트 : 소면은 딱 3분만 삶아야 해요\! 오래 삶으면 퍼져서 맛이 없어요.

code

📋 복사

1\. 물이 팔팔 끓으면 소면 넣기  
   → 소면을 부채처럼 펼쳐서 넣으면 안 뭉쳐요\!  
   → 젓가락으로 바로 한 번 저어주기

2\. 정확히 3분 타이머 맞추기  
   ┌─────────────────────────────┐  
   │  1분 경과 → 젓가락으로 한번 저어주기  │  
   │  2분 경과 → 면 한 가닥 꺼내서 맛보기 │  
   │  3분 경과 → 불 끄고 바로 건지기      │  
   └─────────────────────────────┘

3\. 찬물에 바로 헹구기 (매우 중요\! ⭐)  
   → 찬물에 2\~3번 박박 씻어야 면이 쫄깃해져요  
   → 이 단계 생략하면 면이 퍼지고 뭉쳐요 ❌  
   → 얼음물 사용하면 더 쫄깃\! (선택)

4\. 채반에 올려 물기 제거

   → 손으로 꾹 짜서 물기 최대한 빼기

---

### ⏱️ STEP 4 (4분\~5분) — 비비기 \+ 플레이팅

💡 핵심 포인트 : 양념장은 한 번에 다 넣지 말고 조금씩 넣으면서 간 맞추세요\!

code

📋 복사

1\. 볼에 면 넣기

2\. 양념장 절반 먼저 넣고 비비기  
   → 젓가락으로 면을 들어올리듯 비벼야  
     양념이 골고루 배어요  
   → 간 보고 싱거우면 나머지 양념 추가

3\. 그릇에 담기  
   → 면을 젓가락으로 돌돌 말아서 담으면  
     예쁘게 플레이팅 가능\!

4\. 계란 프라이 올리기

5\. 통깨 솔솔 뿌리기

   → 쪽파, 오이 있으면 함께 올리기

code

📋 복사

└── 텍스트 위주 구성으로 요리 과정이 직관적으로 전달되지 않음

보완 방향  
├── 단계별 사진 또는 일러스트 삽입

## 2️⃣ 독자 신뢰도 확보

"왜 이 레시피를 믿어야 하는가?"

code

기대 효과 → 레시피 신뢰도 상승, 전문성 있는 책으로 포지셔닝 📌
